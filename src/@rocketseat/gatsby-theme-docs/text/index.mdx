# Willkommen

Das komplette **Regelwerk des Scheibensports**, zusammengefasst an einem zentralen Ort!

Have fun, go outside and play 🥏

## Features

- 📝 Offizielle Regeln des [Frisbeesport-Verbands Deutschland](https://frisbeesportverband.de/) für Disc Golf, Ultimate & Freestyle
- 🕵🏻‍♀️ Einfache und schnelle Suche
- 📱 Responsive und mobile friendly
- 🖥 Alle Regeln sind offen und können auf [Github](https://github.com/maxgreive/dfv-rules) bearbeitet und aktualisiert werden
- 🔥 und vieles mehr …
